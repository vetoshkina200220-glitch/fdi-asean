# ASEAN FDI Analytics

Исследование детерминант притока прямых иностранных инвестиций (ПИИ) в страны ASEAN за 2000–2025 годы.

**Цель:** определить, как каждый макроэкономический фактор влияет на приток ПИИ — глобально (панельная регрессия) и в разрезе отдельных стран.

## Целевая переменная

**`Foreign direct investment, net inflows (% of GDP)`** — ПИИ как доля ВВП.

Обоснование выбора (см. `notebooks/00_target_selection.ipynb`):
- Нормализует размер экономики → сравнимость Сингапура и Мьянмы
- Меньшая асимметрия распределения (лучше для OLS)
- Академический стандарт в литературе по детерминантам ПИИ

**Вторичный показатель (robustness check):** `Foreign direct investment, net inflows (BoP, current US$)`

## Признаки по группам

Исходно 17 признаков. По результатам анализа мультиколлинеарности (`01_correlation_analysis.ipynb`) удалён 1 признак: `Mobile_subs` (r = 0.79 с `Internet_users`). В финальных моделях используется **17 признаков**.

Группа «Деглобализация» расширена: вместо ранее используемых `Imports (% GDP)` и `Exports (% GDP)` (которые были компонентами `Trade_openness`) добавлены `Tariff_rate` и `Trade_in_services` — принципиально иные измерения торговой политики и сервисной структуры.

| Группа | Признаки в модели |
|--------|------------------|
| ВВП | GDP annual growth |
| Деглобализация | Trade openness, Tariff rate, Trade in services |
| Декарбонизация | Renewable energy consumption, Energy intensity, CO2 per capita |
| Диджитализация | Internet users %, Secure Internet servers |
| Демография | Population growth, Education expenditure, Labor force participation |
| Долг | General Government Debt, External debt stocks, Total debt service |
| Инфляция | Inflation, consumer prices |
| Процентная ставка | Real interest rate |

## Методология

**Основной метод:** Panel OLS с Fixed Effects (фиксированные эффекты по стране).

Fixed Effects контролирует ненаблюдаемую страновую гетерогенность (институты, история, культура), позволяя оценить «чистый» временной эффект каждого признака. Коэффициенты стандартизованы (z-score) для сравнения силы влияния между признаками с разными единицами измерения.

**Оценка вклада групп:** инкрементальный R² (последовательное добавление группы к базовой модели) + F-тест совместной значимости группы.

## Ключевые результаты

### Глобальная модель (FE OLS, N=175, R² within=0.529)

**Статистически значимые положительные детерминанты (p < 0.01):**

| Признак | β (стандартизованный) | p-value | Группа |
|---------|----------------------|---------|--------|
| Trade_openness | +3.83 | < 0.001 | Деглобализация |
| Edu_expenditure | +1.32 | 0.001 | Демография |

**Статистически значимые отрицательные детерминанты (p < 0.01):**

| Признак | β (стандартизованный) | p-value | Группа |
|---------|----------------------|---------|--------|
| GDP_growth | −1.31 | < 0.001 | ВВП |
| Ext_Debt | −1.11 | < 0.001 | Долг |
| Gov_Debt | −1.08 | 0.004 | Долг |
| Energy_intensity | −0.81 | < 0.001 | Декарбонизация |
| Pop_growth | −0.77 | < 0.001 | Демография |

Новые признаки группы «Деглобализация»: `Tariff_rate` (β = −0.36) и `Trade_in_services` (β = +0.54) имеют ожидаемые знаки, однако не достигают статистической значимости в глобальной модели.

### Вклад тематических групп (ΔR² + F-тест)

| Ранг | Группа | ΔR² (добавление) | F-статистика | Значимость |
|------|--------|-----------------|--------------|-----------|
| 1–2 | Деглобализация | 0.170 | 4.89 | *** |
| 1–2 | Демография | 0.137 | 11.58 | *** |
| 3 | ВВП | 0.093 | 52.87 | *** |
| 4 | Декарбонизация | 0.114 | 1.26 | нет |
| 4 | Долг | 0.107 | 3.85 | ** |
| 6 | Диджитализация | 0.038 | 0.93 | нет |
| 7–8 | Процентная ставка | 0.028 | 0.38 | нет |
| 7–8 | Инфляция | 0.015 | 0.59 | нет |

Группа «Деглобализация» вышла на 1-е место по изолированному ΔR² благодаря расширению с одного до трёх признаков, охватывающих объём торговли, тарифные барьеры и сервисный компонент.

### Страновой анализ (10 стран, Myanmar исключена)

- R² страновых моделей: 0.32 (Бруней) — 0.81 (Камбоджа)
- Иерархическая кластеризация (Ward) выявила **3 группы**:
  - *Кластер 1*: Cambodia, Indonesia, Malaysia, Philippines, Timor-Leste
  - *Кластер 2*: Brunei Darussalam, Lao PDR, Thailand, Viet Nam
  - *Кластер 3*: Singapore — отдельный кластер (финансово-сервисная специализация)

### Pre-COVID vs Post-COVID

| Период | N | R² within |
|--------|---|----------|
| 2000–2019 | 140 | 0.563 |
| 2020–2025 | 35 | 0.950 |

R² within вырос с 0.563 до 0.950 после 2020 года. Ключевые изменения: `Edu_expenditure` значительно усилилась (качество человеческого капитала стало приоритетом для инвесторов), `GDP_growth` и `Pop_growth` стали выраженней в отрицательном направлении.

## Быстрый старт

```bash
# Клонировать репозиторий
git clone <repo-url>
cd fdi-asean-impact

# Создать виртуальное окружение
python3 -m venv .env
source .env/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Запустить JupyterLab
.env/bin/jupyter lab
```

Запускать ноутбуки в порядке нумерации: `00` → `01` → `02` → `03` → `04` → `05`.

> Весь запуск кода только через виртуальное окружение `.env/`. Использование глобального Python/pip недопустимо.
