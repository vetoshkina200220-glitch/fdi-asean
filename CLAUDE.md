# ASEAN FDI Analytics Project

## Цель проекта
Исследование детерминант притока прямых иностранных инвестиций (ПИИ) в страны ASEAN за 2000–2024 годы.
Задача: определить, как каждый фактор влияет на приток ПИИ — глобально (без учёта страны) и в разрезе отдельных стран.

## Данные
**Файл:** `data/asean.xlsx`

**Листы:**
- `Data` — основной датасет (275 строк × 23 колонки): 11 стран, 2000–2024
- `Страны` — справочник стран ASEAN
- `Признаки` — разметка признаков по группам (без is_target — таргет определён явно ниже)

**Страны:** Indonesia, Viet Nam, Singapore, Thailand, Philippines, Cambodia, Brunei Darussalam, Lao PDR, Malaysia, Myanmar, Timor-Leste

## Целевая переменная (TARGET)

**`Foreign direct investment, net inflows (% of GDP)`** — ПИИ как % от ВВП

Выбор обоснован аналитически в `notebooks/00_target_selection.ipynb`:
- Нормализует размер экономики → сравнимость Сингапура и Мьянмы
- Меньшая асимметрия распределения (лучше подходит для OLS)
- Сопоставимая корреляция с признаками — нормализация не теряет сигнал
- Академический стандарт в литературе по детерминантам ПИИ

**Вторичный показатель (robustness check):** `Foreign direct investment, net inflows (BoP, current US$)`

## Признаки по группам
| Группа | Признаки |
|--------|---------|
| ВВП | GDP_annual growth |
| Деглобализация | Trade_openess, Imports (% GDP), Exports (% GDP) |
| Декарбонизация | Renewable energy consumption, Energy intensity, CO2 per capita |
| Диджитализация | Internet users %, Secure Internet servers, Mobile subscriptions |
| Демография | Population_growth, Education expenditure, Labor force participation |
| Долг | General Government Debt, External debt stocks, Total debt service |
| Инфляция | Inflation, consumer prices |
| Процентная ставка | Real interest rate |

## Окружение для выполнения кода

**Виртуальное окружение:** `.env/` в корне проекта

**Правила:**
- Весь запуск кода, установка библиотек и исполнение скриптов — **только** через `.env/`
- Активация: `source .env/bin/activate` перед любым запуском
- Jupyter запускать через: `.env/bin/jupyter notebook` / `.env/bin/jupyter nbconvert`
- Установка пакетов: `.env/bin/pip install ...`
- **Строго запрещено** использовать глобальный Python/pip или менять глобальное окружение системы

## Соглашения по коду
- Весь анализ — в Jupyter notebooks (`notebooks/`)
- Вспомогательные функции — в `src/`
- Графики сохраняются в `outputs/figures/`
- Язык комментариев в коде: русский
- Стандартизация признаков обязательна перед регрессией (для сравнения коэффициентов)
- При построении глобальной модели: не включать страну как признак (задача — глобальная оценка)

## Требования к визуализации
Каждый график обязан иметь:
- **Заголовок** (`plt.title` / `ax.set_title`) — содержательный, на русском языке
- **Подписи осей** (`xlabel`, `ylabel`) с единицами измерения в скобках, например `"ПИИ (% от ВВП)"`, `"Год"`
- **Легенду** (`plt.legend`) при наличии группировок (страны, метрики, группы признаков)
- `plt.tight_layout()` перед `plt.show()` / `savefig()`
- Сохранение в `outputs/figures/` через `savefig(..., bbox_inches='tight')`

Дополнительно:
- Шрифты читаемые: `axes.titlesize=12`, `axes.labelsize=10`, `xtick.labelsize=9`
- Убирать лишние рамки: `spines['top'].set_visible(False)`, `spines['right'].set_visible(False)`
- Цвета стран — фиксированная палитра `tab20`, определённая один раз в начале ноутбука
- Цвета групп признаков — фиксированный словарь `GROUP_COLORS`

## Структура ячеек в Jupyter notebooks
- Логические блоки (раздел анализа) разделяются **Markdown-ячейкой с заголовком и выводом**
- Вывод — короткий (2–4 предложения), но **содержательный**: не "данные выглядят нормально", а конкретный аналитический факт
- Markdown-ячейка нужна **после** логического блока кода — не после каждой ячейки
- Структура раздела: `## Заголовок раздела` (markdown) → код → **`**Вывод:**`** (markdown)
- Финальная markdown-ячейка ноутбука — итоговая таблица ключевых находок

## Интерпретация результатов
- Стандартизованные OLS-коэффициенты: показывают на сколько std изменяется target при +1 std признака
- SHAP: объясняет вклад каждого наблюдения → агрегируем в bar chart
- Отрицательный коэффициент = фактор снижает ПИИ/ВВП

## Ключевые вопросы для аналитики
1. Какие группы факторов наиболее значимы для привлечения ПИИ?
2. Есть ли различия между "зрелыми" (Сингапур, Малайзия) и "развивающимися" (Мьянма, Камбоджа) экономиками?
3. Как пандемия COVID-19 изменила детерминанты ПИИ?
4. Противоречат ли декарбонизация и деглобализация притоку инвестиций?
